## ADR-0004: Max Win + UX Toggles + Anticipation (из отзывов игроков)

**Принято:**
- `MAX_WIN_X = 25000` (теоретическая “мечта”, редкая реализация).
- Post-spin symbol bounce остаётся **по умолчанию ON**, но **обязательны** режимы `Reduce Motion` и `Turbo Spin` (оба по умолчанию OFF).
- “Почти выиграл” делаем **честно**: система Anticipation включается только при реальном математическом шансе, с лимитами частоты и телеметрией.
