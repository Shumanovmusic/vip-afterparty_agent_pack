# Glossary

- **Memory Bank** — набор файлов, которые агент читает как законы.
- **Подплан** — один из 16 разделов работ, который можно вести независимо.
- **Task** — атомарная задача с критериями приёмки.
- **Идемпотентность** — повтор одного и того же запроса не создаёт дубли и не ломает состояние.
