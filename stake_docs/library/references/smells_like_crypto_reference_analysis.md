# Smells Like Crypto — Reference Analysis (Hacksaw Gaming, Stake)

Этот документ — локальная копия референс-анализа UX (тайминги, эффекты, автоспин, BOOM, dimming, win popups).
Используется как ориентир для SCENARIO_V1.md и UX_ANIMATION_SPEC.md.

Ключевые паттерны:
- Быстрый цикл spin->stop->result
- Мгновенные win popups прямо на поле (+0.5)
- Dimming невыигрышных символов
- BOOM/комикс FX на выигрышах
- Простое меню автоспинов (10/50/100) и подтверждение старта
